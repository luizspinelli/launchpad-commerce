## 📊 Key Decisions (ADRs)

### Why Next.js?
**Status:** ✅ Decided  
**File:** `decisions/001-next-js-selection.md`

**Decision:** Use Next.js 14 with App Router

**Rationale:**
- ISR + SSR native (landing page performance)
- Built-in API routes (Stripe webhooks)
- Vercel first-class support (zero-config deployment)
- TypeScript by default
- Edge functions (future scaling)

---

### Why Stripe?
**Status:** ✅ Decided  
**File:** `decisions/002-stripe-integration.md`

**Decision:** Stripe for payments

**Rationale:**
- PCI Level 1 compliance (security)
- Webhook support (order processing)
- Multiple payment methods out-of-box
- Excellent docs & community
- Free test mode

---

### Deployment: Vercel
**Status:** ✅ Decided  
**File:** `decisions/003-deployment-vercel.md`

**Decision:** Deploy to Vercel

**Rationale:**
- Zero-config for Next.js
- Automatic CI/CD on git push
- Free tier sufficient (100GB bandwidth, unlimited deployments)
- Edge functions if needed
- One-click rollbacks
